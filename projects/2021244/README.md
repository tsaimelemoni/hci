# ΕΙΣΑΓΩΓΉ

Οι στόχοι μου για το μάθημα είναι να εμβαθύνω τις γνώσεις μου για την σχέση που έχουν οι άνθρωποι με τον υπολογιστή και μέσω του github να αναπτύξω τις ικανότητες μου στην  ομαδικότητα.  

Όλα τα μελλοντικά μου προγράμματα θα έχουν ως στόχο την μεγαλύτερη αλληλεπίδραση μεταξύ Ανθρώπου-Υπολογιστή, αλλά και την άριστη συνεργασία των μελών της ομάδας μου, οπώς κατά άποψη μου θα έπρεπε να έχει κάθε άνθρωπος που ασχολείται με αυτό το επάγγελμα.
